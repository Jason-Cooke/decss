Introducing Decss

A (mostly) CSS-driven presentation framework

Decss: Present from the Web, Beautifully

  • Uses CSS3 for transitions
  • Built with SASS for easy customization
  • Sync system for remote follow-along
  • Responsive, mobile-friendly layouts
  • Flexbox for automagic content layout
  • Designed from the ground up for presenters

Presenter Mode

  • Presentation timer
  • Jump to any slide
